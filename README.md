# Todo list Test task

Тестовое задание для компании UITOP. Приложение разработано с использованием React (Vite + TypeScript), Express, SQLite и чистого CSS.

## Функционал
- Создание задач с привязкой к категориям.
- Ограничение: **не более 5 активных задач** в одной категории (валидация на бэкэнде).
- Мягкое завершение и удаление задач через механизм **Undo (отмена)** в течение 5 секунд.
- Фильтрация задач по категориям.

## Инструкция по локальному запуску

### 1. Запуск Сервера (Backend)
```bash
cd backend
npm install
npm run dev
```

### 2. Запуск сайта (Frontend)
```bash
cd frontend
npm install
npm run dev
